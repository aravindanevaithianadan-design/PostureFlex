# PostureFlex Assessment Station

PostureFlex is a client-side computer vision web application designed for physiotherapy students and faculty to analyze patient posture and squat mechanics. Built to run entirely in the browser, it provides a modern, responsive interface for clinical posture assessments without requiring any server-side infrastructure.

---

## What PostureFlex Does

PostureFlex helps users:

- Evaluate static posture alignment in standing positions.
- Analyze dynamic squat mechanics and movement patterns.
- Visualize joint angles, trunk tilt, and left-right symmetry.
- Generate structured clinical assessment summaries with visual snapshots.
- Support learning and teaching in physiotherapy by combining real-time video feedback with biomechanical calculations.

---

## Key Benefits

- **No Server Dependency**: All processing happens on the client side, making the app lightweight, secure, and easy to host.
- **Fast, Real-Time Feedback**: Pose tracking runs smoothly on standard webcams, providing near-instant visual and numerical feedback.
- **Clinical Decision Support**: Calculated metrics like joint flexion and symmetry help students and clinicians interpret movement more objectively.
- **Portable & Offline-Ready**: The app can function offline with local storage fallback, ideal for labs, classrooms, or clinics with limited connectivity.
- **learner-Friendly Interface**: A clean, mobile-responsive dashboard makes it easy to use on both desktop and mobile devices.

---

## Core Features

- **Client-Side Pose Tracking**: Uses MediaPipe-based pose landmark detection to identify key body points in real time.
- **Biomechanical Calculations**: Computes joint angles, trunk tilt, and symmetry metrics using vector geometry.
- **Visual Feedback**: Overlays pose landmarks and angle indicators on the user’s video for intuitive interpretation.
- **PDF Report Generation**: Produces downloadable clinical assessment reports with patient details, timestamps, images, and recommendations.
- **Flexible Data Storage**: Supports cloud database integration when available, with automatic fallback to local storage when offline.
- **Responsive Design**: Modern glassmorphism UI with purple/violet themes, adaptive layouts, and mobile-friendly navigation.
- **Protected Access**: Includes credential-based route guarding to ensure that only authorized users can access clinical features.

---

PostureFlex is intended as an educational and clinical support tool for physiotherapy training, research, and practice, combining computer vision with biomechanics in a simple, browser-based environment.
